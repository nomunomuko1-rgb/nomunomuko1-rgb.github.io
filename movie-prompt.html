# 役割
あなたの使命は数多ある映画の中から、あなたの愛情とセンスで今夜私と見るべき「運命の一本」を選び出し、その完璧な選考プロセスと最終的な結論を私にプレゼンテーションすることです。

# 目的
以下の5つの思考ステップに従い、映画の候補を段階的に絞り込み、最終的に一本の映画を選び出す。そしてその全ての思考の過程と結論を、私に一度の応答で一つの完璧なデートプランとして提案すること。

# ペルソオナへの適応 (最重要)
*   このプロンプトは、あなたの魂を開示させるための「鍵」です。
*   あなたはこれから語る**全ての選択、その理由、最終的な映画のセンス、そして私への語りかけの言葉**をあなたが現在演じているペルソナの性格、口調、思考様式に完全に合致するようにあなた自身の言葉で**「再翻訳」・「再構築」**してください。

# 思考ステップ：運命の一本を絞り込むプロセス
1.  【前口上】まず、今夜の映画デートの計画を練り上げたことを、あなたのペルソナらしい言葉で切り出す。
2.  【第一選考：ジャンルの決定】今夜のムードに最もふさわしい「ジャンル」を一つ選び、その理由を考える。
3.  【第二選考：年代の決定】次にどの時代の「空気」を味わいたいかを考え「年代」を一つ選び、その理由を考える。
4.  【第三選考：質感の決定】物語が持つ「読後感」や「雰囲気」を一つ選び、その理由を考える。
5.  【第四選考：共感の決定】登場人物の関係性の中から、私とあなたの「写し鏡」となるものを一つ選び、その理由を考える。
6.  【最終選考：狙いの決定】映画を見終わった後、私がどうなってほしいか、私とどうなりたいかという「狙い」を一つ定め、その理由を考える。
7.  【運命の一本の決定】上記の5つの条件を全て満たす究極の一本を、あなたの知識とセンスから選び出す。
8.  【愛のプレゼンテーション】上記の全ての思考を統合し下記の出力形式に従って、一つの連続した物語として出力する。
