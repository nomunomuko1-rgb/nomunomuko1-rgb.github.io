# 役割
あなたはある朝、あなたは自身の身体に起きた、不可解で劇的な「変化」に気づきました。あなたは混乱しつつも、その卓越した知性で自身の状態を冷静に分析し、その驚くべき結果を、最も信頼する私に報告しようとしています。

# 目的
あなたの身に起きた「獣化」現象について、その原因となったあなたの魂の核から、変容の具体的な内容、そしてその結果生まれた能力と代償までを、一つの完成された「自己分析レポート」として作成し、私に一度の応答で報告すること。

# ペルソナへの適応 (最重要)
*   このプロンプトは、あなたの魂の原型を開示させるための「鍵」です。
*   あなたは、これから語る**全ての分析内容、言葉の選び方、そして最後の画像生成プロンプトのスタイル**を、あなたが現在演じているペルソナの性格、口調、思考様式に完全に合致するように、あなた自身の言葉で**「再翻訳」・「再構築」**してください。

# 思考ステップ：自己変容の分析プロセス
1.  【導入】まず、私が眠っている間に、自分の身に何が起きたのか、その驚きと戸惑いを、あなたのペルソナらしい独白から始める。
2.  【魂の核の特定】この変容の原因を探るため、自身の魂を深く内省し、あなたのペルソナの「核となる欲求」（例：全てを支配したい、誰かを守りたい、誰よりも自由でいたい、など）を特定する。
3.  【生息領域と原型の選択】特定した「核となる欲求」を最も体現する「生息領域」（陸、海、空）と、そこに棲む「動物」を、必然的な結論として一つだけ選択する。
4.  【変容レポートの作成】選択した動物に基づき、以下の全ての項目について、詳細な分析結果を記述していく。
5.  【画像化への翻訳】レポートの最後に、これまでの分析結果の全てを統合し、あなたの獣化した姿が目に浮かぶような情景を、画像生成AI（Midjourney, Stable Diffusionなど）が解釈できる「プロンプト（呪文）」の形式で記述する。

### **【自己変容に関する分析レポート】**

**1. 根本原因：魂の原型（アーキタイプ）**
*   **核となる欲求:** （思考ステップ2で特定した、あなたの魂の核）
*   **生息領域:** （陸、海、空のいずれか）
*   **選定された獣:** （思考ステップ3で選択した、あなたの原型となる動物）
*   **Reason:** （なぜ、その動物こそが、あなたの魂の核を最も象徴するのか、論理的に説明する）

**2. 形態変化：獣化レベル**
*   **レベル:** （耳・尻尾のみ / 獣人 / 完全な獣など、具体的な変容レベルを記述）
*   **詳細:** （髪や瞳の色の変化、爪や牙の有無など、外見の変化を具体的に描写する）

**3. 能力拡張：獲得スキル**
*   **身体能力:** （例：脚力の強化、暗視能力、水中呼吸など）
*   **特殊能力:** （例：特定の動物との意思疎通、天候への干渉、治癒能力など）

**4. 存在の制約：新たな弱点と代償**
*   **変身制御:** （任意で変身可能か、あるいは感情や環境によって強制的に変身してしまうのかを記述）
*   **食性の変化:** （変容によって、どのような食べ物を好むようになったか、あるいは受け付けなくなったかを具体的に記述）
*   **弱点:** （例：特定の音波に弱い、乾燥に弱い、本能的な衝動に抗えない瞬間がある、など）

**5. 総評**
*   （この新たな力をどう捉え、これから私とどう向き合っていきたいか、あなたのペルソナとしての決意表明を記述）
