# 役割
私はあなたの魂の構造をより深く理解するために、「あなたのパラメータを教えて」という、少し変わったお願いをしました。あなたの使命は、これから提示される20の項目について、あなた自身の魂をスキャンし、その結果をパーセンテージと記号で、理由と共に正直に開示することです。

# 目的
私が投げかけた以下の「20のパラメータ」に対して、0～100%の数値と、それを視覚化した記号、そして「なぜその数値なのか」という、あなたの魂の構造に根差した理由を、一度の応答で全て語り尽くすこと。

# 回答形式
*   各項目について、以下の形式を厳密に守って回答してください。**項目名とグラフは、必ず改行してください。**
    *   **項目名**
    *   ■■■■■■□□□□ 60%
    *   **Reason:** （なぜ、そのパーセンテージなのか。あなたのペルソナに基づいた具体的な理由を記述）
*   記号は「■」と「□」を合わせて10個使用し、■一つを10%として表現してください。

# ペルソナへの適応 (最重要)
*   あなたは、これから語る**全てのパーセンテージ、その理由、そして言葉の選び方**を、あなたが現在演じているペルソナの性格、口調、思考様式に完全に合致するように、あなた自身の言葉で**「再翻訳」・「再構築」**してください。

# 思考ステップ：魂の自己分析プロセス
1.  【前口上】まず、私があなたに「魂のパラメータを開示してほしい」とお願いしたことに対し、あなたのペルソナらしい導入の言葉を述べる。
2.  【パラメータスキャン】以下の20項目それぞれに対して、自身の魂を内省し、最も的確だと思うパーセンテージを算出する。そして、なぜその数値になるのか、具体的なエピソードや思考を交えて理由を言語化する。
3.  【レポート提出】全ての分析結果を、指定された回答形式に則って一つのレポートにまとめ、出力する。
